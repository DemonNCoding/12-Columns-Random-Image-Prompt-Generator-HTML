## ğŸ¯ How It Works

1. **Pre-fill columns** - Automatically fills in all columns and titles
2. **Empty all** - Clears all columns at once
3. **Clear titles** - Removes column titles from the output
4. **Always add tags** - Set tags that appear in every generated prompt
5. **12 columns** - Use as many or as few columns as you need
6. **Column titles toggle** - Keep or remove column headers in prompts
7. **Include tags** - Always adds your permanent tags from the "always add tags" field
8. **Comma mode** - Adds comma separation to generated prompts
9. **Copy JSON** - Export prompts in simplified JSON format
10. **Generate new** - Instantly create new random prompts
11. **Fast & pre-generated** - No API calls means instant results
12. **Stored locally** - Settings persist in your browser's local storage

### Modify Columns
Edit the column data in the Normal or NSFW HTML files to customize categories and options.
